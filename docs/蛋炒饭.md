# 蛋炒饭

## 材料

剩米饭

两个鸡蛋

## 烹饪

1. 将米饭碾压成松散状态
2. 1/3蛋液和米饭搅拌，剩下鸡蛋液备用
3. 冷锅下一大勺油，油冒烟后下2/3蛋液，快速炒成性
4. 把鸡蛋推到一边，开小火，在锅边倒入一勺酱油。等酱油开始冒大泡，将鸡蛋和酱油搅拌均匀
5. 火调到最大，把米饭倒入锅中，用铲子边压边炒米饭，让米粒颗颗分明
6. 将大鸡蛋碾压成小块，加入一勺盐，继续翻炒米粒跳动，即可出锅
